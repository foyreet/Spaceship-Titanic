# Spaceship-Titanic
https://www.kaggle.com/competitions/spaceship-titanic/overview

Разработал стекинг-модель на базе Random Forest, Gradient Boosting и Logistic Regression для решения задачи бинарной классификации (точность на кросс-валидации ~80.4%).
Выполнил Feature Engineering: обработал пропуски, извлек данные о палубе и борте из номеров кают, а также сгенерировал признаки размера семьи и групповой принадлежности
